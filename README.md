## Requisitos

```bash
Composer
PHP >= 8
```

## Variáveis de ambiente

```bash
Configurar o arquivo .env com as credenciais do seu banco de dados
```

## Instalando dependências
```bash
composer install
ou
composer update 
```

## Migrations
```bash
php artisan migrate 
```
## Iniciando servidor
```bash
php artisan serve
```
